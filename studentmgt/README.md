The Student Attendance Management System is a Java-based application that allows administrators to manage student attendance records efficiently. This system enables users to:

Add new students
Mark attendance (Present, Absent, Late)
View student and attendance records
Generate monthly attendance reports
The project follows an MVC (Model-View-Controller) pattern and uses MySQL as the database.
